# LCD Base Station Front Plate

- 12x capacitive buttons
- Space for 128x64 LCD
- Connection to base station

## Buttons

### LCD

- Top Left
- Bottom Left
- Top Right
- Bottom Right

### Directional

- Up
- Down
- Left
- Right

### General

- Home
- Menu
- Enter
- Cancel
