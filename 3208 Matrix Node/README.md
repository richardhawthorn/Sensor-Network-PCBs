# 3208 Matrix Node

- 3208 matrix (Sure Electronics)

## Other Features

- ATMEGA328P with Arduino boot-loader
- RMF12B/RFM69W radio
- 3.3V operation
- Mini USB power connector
- FTDI programming port
- ICSP port
- LED output
