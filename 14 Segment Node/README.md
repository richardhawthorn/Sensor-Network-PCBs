# 14 Segment Node

- 0.54" 14 segment display, 4 character
- HT16K33 i2c LED driver

## Other Features

- ATMEGA328P with Arduino boot-loader
- RMF12B/RFM69W radio
- 3.3V operation
- Mini USB power connector
- FTDI programming port
- ICSP port
- LED output
