### Features

- Prototype area for your own circuits
- ATMEGA328P with Arduino boot-loader
- RMF12B/RFM69W radio
- 3.3V operation
- Mini USB power connector
- Li-po charger
- Battery charging LED
- Automatic battery/USB power switching
- JST Battery connector
- FTDI programming port
- LED output
